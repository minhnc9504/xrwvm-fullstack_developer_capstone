# Full Stack Development Capstone Project

## Project Name: Best Cars Dealership - National Car Retailer

### Project Description:
This is a full-stack web application developed for "Best Cars Dealership", a car retailer in the U.S. 
The application allows users to:
- View dealership branches across the U.S.
- Filter dealers by state.
- View and submit reviews for car dealers.
- Perform sentiment analysis on user reviews.

### Technologies Used:
- **Frontend:** React, HTML, CSS, Bootstrap.
- **Backend:** Django, Flask (for Microservices).
- **Database:** MongoDB (Dealers/Reviews), SQLite (Car Models).
- **Deployment:** Docker, Kubernetes, IBM Cloud.
- **CI/CD:** GitHub Actions.
